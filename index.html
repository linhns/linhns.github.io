FlashBack
Jan - May, 2021

FlashBack is a flashcard app designed for students to improve memory retention via active recall. It is CLI-esque to cater to fast typists. This was made as the team project for the Software Engineering module in NUS.

Technologies: Java, Markdown
NextWordPredictor
May - Aug, 2020

NextWordPredictor is a word suggester based on user input. This was built as the capstone project for the John Hopkins University Coursera Data Science course.

Technologies: R, RMarkdown
